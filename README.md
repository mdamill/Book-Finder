# Alex's Book Finder

A book discovery app built with React and styled using TailwindCSS. It integrates with the Open Library API to help users find books by it's title.

## Tech Stack

- **Frontend**: React
- **CSS Framework**: TailwindCSS
- **API**: Open Library API
